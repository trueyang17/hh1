1、常用单位

  • 1、px 像素

  • 2、em 相对字体

    • em是相对长度单位。相对于 当前对象内文本的字体尺寸。
      • 1em = 当前对象文本字体的大小
      • 如当前对行内文本的字体尺寸未被人为设置，则相对于浏览器的默认字体尺寸。
    • 谷歌浏览器默认的字体大小为16px

  • 3、rem
    rem r->root

  • html是html的根元素
  • rem：是根据html的font-size大小来变化的，默认情况下我们的字体大小为16px
  • 1rem = html字体的大小

2、CSS3新增背景属性

  • background-clip 背景裁切（背景绘制区域）
    • border-box(默认值)背景是从边框部分开始绘制的
    • padding-box背景从padding部分开始绘制的
    • content-box背景从内容部分开始绘制
  • background-origin 背景图片定位区域
    • border-box 背景从border部分开始定位绘制
    • padding-box 默认值，规定背景从padding部分

4、CSS Sprite雪碧图

1.什么是CSS Sprite

CSS Sprites也叫css精灵，是一种网页图片应用处理方式。它是将所有被用到的小图片全部放在一张大图中，然后用背景定位属性进行控制要显示的图片，这样加载网页的时候只需对图片请求一次即可。

2.为什么需要使用CSS Sprite雪碧图

因为一个网页中，往往包含许多小图标，而这些小图标都是一个个image组成的，当我们加载网页的时候，如果这个页面有一百个小图标，那么浏览器就需要向服务器发送一百次http请求，因为每请求一次就会和服务器链接一次，耗时严重，对用户访问网页不友好，这时如果考虑将小图片合并成一张大图

3.实现原理

  • 利用background-position在一张大图中定位裁剪出我们需要的一部分，然后用这个部分当做单个的图片使用。
  • 它是以图片的左上角为圆点坐标，两个参数分别是x，y轴坐标起始点，

4. 什么时候用雪碧图

  1. 静态图片不随用户信息的变化而变化
  2. 小图片，图片容量比较小
  3. 加载量比较大
    • 一些大图不建议拼成雪碧图

5. 用雪碧图的优点

  • 有效减少http请求数量，加速内容显示。因为每请求一次，就会和服务器链接一次，建立链接是需要额外时间的

6.Tips

  • 制作雪碧图时，建议先想好每个小图标的大小再制作雪碧图，因为在css中调整一个图片的大小并不是很方便。
  • 在线制作雪碧图时，会自动生成代码，在实际体验中，这些代码可以供我们参考，实际应用中还需要我们自己做细微的调整
  • 调整时，利用chrome下的控制台去做细微调整

5、css命名规范

  • 必须使用英文开头,并且开头字母一律小写;
  • 所有的命名最好都小写
  • 尽量不要用缩写英,除非可以一目了然的;
  • 如果遇到相差不大class或者id，主功能识别字母在前，位置识别字母在后，位置识别字母
    第一个可大写（如：navTop，menuLeft）
  • 遵循驼峰命名法:第一个单词的首字母小写,其余每一个有意义的单词的首字母大写（如：
    studentInfo、getElementById）
