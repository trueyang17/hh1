网页

通俗来讲我们的网页就是html页面，一个html文件就相当于一个网页；网页是构成网站的基本要素，需要通过浏览器来预览

网站

网站是由很多个网页组成的，共同为一个目标来服务。

站群（流水线统一化管理）

站群是由很多个网站组成的，共同为一个公司服务。
例如：百度

  • 百度糯米、百度音乐、贴吧、百度金融

html标签

由开始标签和结束标签组成

  • 开始标签 <html>小于号 + 标签名 + 大于号
  • 结束标签</html> 小于号+ 斜杠+标签名+大于号
  • html它叫根元素

meta标签

  • 单标签 <meta /> 小于号 + 标签名 + 斜杠 + 大于号
  • 属性charset="utf-8"
<meta charset="utf-8" />



meta:标签名

charset="utf-8" 标签属性

charset 标签属性名

utf-8 标签属性值 属性值用双引号引起来

title

显示网页的标题

快速生成标签方式

  • 标签名 + tab

html元素

html元素通常由开始标签、结束标签以及标签之间的内容组成

  • 常规元素 <title>网页标题</title>
    • 常规元素可以嵌套其他的元素
  • 空元素 例meta标签，一般用来定义功能
    • 空元素不能实现嵌套功能

html属性

属性一般写在我们的开始标签的标签名后面
属性名和属性值之间用=链接
一个元素可以有多个属性

  • 属性组成部分
    • 标签属性名
    • 标签属性值（属性值用双引号包裹）
一个元素可以有多个属性

<div class="div1" title="我是div">

</div>
