1、CSS Background（背景）

1）background-color 背景颜色

background-color : 设置或检索对象的背景颜色，默认值：transparent；
同时定义了背景颜色和背景图像时，背景图像覆盖在背景颜色之上

  • 属性值有三种书写方式
    • 第一种：关键字法（不常用）
    • 第二种：十六进制法 常用
    • 第三种：rgb(0~255,0~255,0~255)

2）background-image背景图片

  • 语法background-image:url("图片路径")
  • 特点：铺满整个内容区，并且x轴和y轴重复平铺
  • background-image: 设置或检索对象的背景图像，默认值：none；
  • 如果设置了 background-image，同时也建议设置 background-color 用于当背景图像不可见时保持与文本颜色有一定的对比度。
  • 如果定义了多组背景图，且背景图之间有重叠，写在前面的将会盖在写在后面的图像之上。

3）background-repeat背景重复

  • 功能：设置或检索对象的背景图像如何铺排填充。必须先指定 background-image 属性。默认值：repeat；
  • 允许提供2个参数，如果提供全部2个参数，第1个用于横向，第二个用于纵向；

常用属性值及意义

属性值 描述
repeat 默认。背景在水平和垂直方向平铺
repeat-x 背景在水平方向进行平铺
repeat-y 背景在垂直方向进行平铺
no-repeat 不平铺，只显示一张图片背景

4）background-position背景定位

必须先指定 <’ background-image ‘> 属性。 该属性提供2个参数值

  • 语法：background-position:x y;
  • 数值表示方式：
      1. 可以为具体数值
      1. 可以为关键字top、left、right、bottom
    • 3.可以设置为百分比
  • 当我们只设置一个值的时候，该值将用于横坐标；纵坐标默认为50%（即center）

5）background-attachment 背景关联

  • 设置或检索背景图像是随对象内容滚动还是固定的。必须先指定background-image属性。默认值为：scroll
属性值 描述
scroll 默认，背景图像随着我们页面的其余部分滚动
fixed 背景图像固定

6）background背景复合属性

  • 语法：background: color image position repeat attachment
**背景复合属性**

 background: #f00 url("img/bd_logo.jpg") center center no-repeat scroll;

 /*背景颜色 背景图像 位置 图像是否重复 背景关联*/

background-size 背景图片的大小

- 属性值可以为：

    - 具体的数值

        - 当只设置一个值的时候，表示的是宽度，高度自适应（保持宽高比例进行缩放）

    - 百分比（相对我们的容器）

    - auto 默认值，背景图像的真实大小

    - cover 将背景图像等比缩放到完全覆盖容器，背景图像可能超出容器。

    - contain 将背景图像进行等比缩放，背景图像始终被包含在我们的容器里

2、PS工具的使用

1.打开设计稿

把设计稿直接拖到PS工具中，然后松开即可
文件 => 打开 ctrl + O
2.关闭设计稿

直接点击设计稿面板对应的关闭按钮 X
文件 => 关闭 ctrl + W
3.缩放设计稿

ctrl +/-
alt + 鼠标滚轮
ctrl + 1 回到设计稿100%的状态
ctrl + 0 回到设计稿打开的初始尺寸
4.给设计稿解锁和上锁

解锁: 双击设计稿被锁住对应的图片
上锁：图层面板有一个小锁的小图标
5.拖动设计稿

按住空格键 (鼠标就会出现一个小手的形状 ) ,鼠标拖动设计稿
6.回到设计稿原始的状态

窗口 => 历史记录 （记录了每一步对设计稿的操作）
7.取消虚线框

ctrl + D
8.快速切出我们想要的素材图片

在对应的图层上 => 鼠标右键 => 智能对象（图层右下角多了一个小缺角 - 智能对象缩览图）=> 双击这个智能对象缩览图
9.保存我们切下来的图片

文件 => 存储为web所用的格式
ctrl + alt + shift + S
