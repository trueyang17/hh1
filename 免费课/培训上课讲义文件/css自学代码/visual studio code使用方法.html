Visual Studio Code 有两个比较重要的文件

  • setting.json （基本设置）
    • 【文件】-【首选项】-【设置】
  • keybindings.json（配置快捷键文件）
    • 【文件】-【首选项】-【键盘快捷方式】

常用快捷方式

  • Alt + B 用默认浏览器打开
  • Ctrl + J 调出控制台 js部分用
  • Ctrl + W 关闭窗口
  • Ctrl + B 切换侧边栏可见性
  • Ctrl + / 行注释
  • Ctrl + Del 使代码回退到光标处
  • Ctrl + Shift + K 删除行
  • Ctrl + 右箭头 光标向前移动
  • Ctrl + 左箭头 光标向后移动
  • Shift + Alt + 向上向下箭头 复制此行
  • Ctrl + Alt + PageUp 向上滚动（行）
  • Alt + 向上箭头 向上移动行
  • Alt + 向下箭头 向下移动行
  • Ctrl + Shift + Enter 在上面插入行
  • Ctrl + Alt + 向上箭头 在上面插入光标
  • Ctrl + Alt + 向下箭头 在下面添加光标
  • Ctrl + Shift + H 在文件中替换
  • Ctrl + Shift + F 在文件中查找
  • Ctrl + X 剪切当前行或者选中的代码
  • Ctrl + C 复制
  • Ctrl + V 粘贴
  • Ctrl + O打开文件
  • Ctrl + = 放大
  • Ctrl + - 缩小
  • Ctrl + S 保存
  • Ctrl + Shift + N新建窗口
  • Ctrl + H 替换
  • Ctrl + F 查找
  • Shift + Alt + F 格式化文件
  • Ctrl + K \ Ctrl + \正交拆分编辑器
  • Ctrl + End 滚动到底部
  • Ctrl + Home 滚动到顶部
  • Ctrl + Tab 在文件之间切换
  • Ctrl + G 跳到指定行
    • Shift + Alt + 0 切换垂直/水平编辑器布局
