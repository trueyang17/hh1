一、transform

1、 transform 并不是一个动态属性，是一个静态的属性。
2、 专注于改变元素基于默认位置的——位置变化、大小变化、转动以及拉伸。

1. transform属性浏览器的支持情况

  • IE 10 Firefox google 和Opera支持transform
  • Safari要求前缀 -webkit-

  • IE9要求前缀-ms-

  • transform书写情况

    • transform : 2D转换方法
    • -ms-transform : 2D转换方法/IE 9/
    • -webkit-transform：2D转换方法/safari /

2.translate(x,y)

  • 定义 2D 转换，沿着 X 和 Y 轴移动元素。
  • translateX(n) 定义 2D 转换，沿着 X 轴移动元素。
  • translateY(y) 定义 2D 转换，沿着 Y 轴移动元素。

3.scale(x,y)

  • 定义 2D 缩放转换，同时改变元素的宽度和高度。
  • scaleX(n) 定义 2D 缩放转换，改变元素的宽度。
  • scaleY(n) 定义 2D 缩放转换，改变元素的高度。

scale(2, 3) 意思为转变宽度为原来大小2倍，高度为原来的三倍

4.rotate(angle)

  • 定义 2D 旋转，在参数中规定角度。
  • 语法 rotate(30deg)
    • rotate(30deg)意思为顺时针旋转30度
    • 单位deg表示角度，这里只设置一个值，值为正表示顺时针旋转多少度，值为负数表示逆时针旋转多少度
    • 默认情况是围绕元素中心点发生旋转

5.skew(x-angle,y-angle)

  • 定义 2D 倾斜转换，沿着 X 和 Y 轴发生倾斜。
  • skewX(angle) 定义 2D 倾斜转换，沿着 X 轴。
  • skewY(angle) 定义 2D 倾斜转换，沿着 Y 轴。

6.transform-origin （转换属性）

允许转换元素的基点位置

  • 语法： transform-origin: x y;

可以为关键字top left bottom right 数值 和 百分比
2D中基点默认为元素中心点（50% 50%）

二、transition CSS3过渡属性

transtion是指当它所绑定的属性（比如width height）发生改变的时候，不再突变，而是根据速度曲线变化
语法：transition: property duration timing-function delay;

属性 值
transition 复合属性，用于在一个属性中设置四个过渡属性。
transition-property 规定应用过渡的 CSS 属性的名称。
transition-duration 定义过渡效果花费的时间。默认是 0。
transition-timing-function 规定过渡效果的时间曲线。默认是 “ease”。
transition-delay 规定过渡效果何时开始。默认是 0。
  • transition-property
    • 语法：transition-property : none | all | property;
值 描述
none 没有属性会获得过渡效果
all 所有属性都将获得过渡效果
property 定义应用过渡效果的CSS属性名称列表，列表用逗号分割
  • transition-timing-function
    • 语法：transition-timing-function: linear|ease|ease-in|ease-out|ease-in-out
值 描述
linear 规定以相同速度开始至结束的过渡效果
ease 规定慢速开始，然后变快，然后慢速结束的过渡效果
ease-in 规定以慢速开始的过渡效果
ease-out 规定以慢速结束的过渡效果
ease-in-out 规定以慢速开始和结束的过渡效果

三、animation CSS3动画

  • 概念：动画是使元素从一种样式逐渐变化为另一种样式效果。
  • 作用及特点
    • CSS3动画可以取代许多网页动画图像
    • 可以改变任意多的样式和任意多的次数

1.@keyframes 创建动画

  • 用百分比来规定变化发生的时间，或用关键词”form”和”to”，等同于0%~100%。 0%是动画的开始，100%是动画的完成。

  • 为了得到最佳的浏览器支持，我们应该始终定义0%~100%的动画

  • 语法：如下

@keyframes myfirst{

    0%   {background: red;}

    25%  {background: yellow;}

    50%  {background: blue;}

    100% {background: green;}

}



@-webkit-keyframes myfirst /* Safari 与 Chrome */{

    0%   {background: red;}

    25%  {background: yellow;}

    50%  {background: blue;}

    100% {background: green;}

}

此时只是定义了动画的规则，并没有执行动画，需要将动画绑定到一个选择器

2. animation复合属性

keyframes只是创建动画的过程 ，我们怎么让他执行呢？
1、需要绑定到一个选择器利用animation去触发这个动画效果
触发动画至少需要两个属性值

  • 动画的名称
  • 动画的时长
    语法
    animation : name duration timing-funtion delay iteration-count direction;
实例

把"myfirst"动画绑定到div元素，时长：5s;

div{

    animation:myfirst 5s;

    -webkit-animation:myfirst 5s;/*safari 与chrome*/

}

我们必须同时定义动画的名称和动画的持续时间。
animation 属性是一个复合属性，用于设置六个动画属性

属性 描述
animation-name 规定@keyframes动画的名称
animation-duration 规定动画完成一个周期的时间。默认0
animation-timing-function 规定动画的速度曲线(运动方式)。默认是”ease”
animation-delay 规定动画何时开始。默认0
animation-iteration-count 规定动画被播放的次数。默认1
animation-direction 规定是否应该轮流反向播放动画

animation-iteration-count 动画运行的次数
-语法：animation-iteration-count: value;

  • value为具体的数值
  • infinite 指定动画应该播放无限次（永远）

steps
animation默认以ease方式过渡，它会在每个关键帧之间插入补间动画，所以动画效果是连贯性的，除了ease，linear函数都会为其插入补间。但是有些效果不需要补间，只需要关键帧之间的跳跃，而不是从一种状态持续到另一种状态的过渡，这是就应该用steps

参数是一个正值，指定我们希望动画分割的段数
